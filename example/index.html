About

This is a JavaScript plug-in that can encode and decode characters with Base64.

This plug-in can be used for characters that UNICODE is between 0000 0000 and 0000 FFFF. Include Number, English, Chinese, Korean and so on.

And you can set the Secret Key by your own situation.

Usage

To use codeBase64, the following files should always be included.

<meta http-equiv="Content-Type" content="text/html; charset=UTF-8"/>
<script type="text/javascript" src="codeBase64.js"></script>

Suppose we want to encode a string, then decode it. If you do not fill Secret Key, the default is null. Result is what that encode or decode your input directly. CodeBase64 makes it as simple as:

var result = codeBase64.encode('hello'[,headKey,endKey]);
var str = codeBase64.decode(result[,headKey,endKey]);
